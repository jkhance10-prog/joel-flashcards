# Joel's Flashcards 📚

A modern, clean web app for studying flashcards.

## Features
* **Dark/Light Mode** Toggle
* **3D Card Tilt** on hover
* **GlowingHeader Text**

## 🛠️ Tech
HTML, Tailwind CSS, Vanilla JS.
